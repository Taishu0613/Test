# どうせ覚えるショートカット

| キーボード               | できる事                   |
| :----------------------- | :------------------------- |
| `ctrl`+`F`               | 任意の文字列検索           |
| `範囲選択`+`tab`         | 同時ににインデントを増やす |
| `範囲選択`+`shift`+`tab` | 同時ににインデントを減らす |
| `範囲選択`+`alt`+`/`     | 同時にコメントアウト       |
| `ctrl`+`Z`               | 一つ前に戻る               |
| `ctrl`+`C`               | コピー                     |
| `ctrl`+`V`               | ペースト                   |
| `ctrl`+`X`               | 切り取り                   |
| `shift`+`caps`           | 大文字小文字入力反転       |
